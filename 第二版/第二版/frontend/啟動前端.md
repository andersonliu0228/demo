# 前端啟動指南

## 快速啟動（3 步驟）

### 1. 安裝 Node.js 依賴

```powershell
cd frontend
npm install
```

這會安裝所有必要的依賴包（React, Vite, Tailwind CSS, Axios 等）。

### 2. 確認後端運行

確保後端 API 正在運行：
```powershell
# 在專案根目錄
docker ps
```

應該看到 `ea_trading_backend` 容器正在運行。

### 3. 啟動前端開發伺服器

```powershell
npm run dev
```

前端將運行在：**http://localhost:5173**

## 訪問應用

1. 打開瀏覽器訪問：http://localhost:5173
2. 使用測試帳號登入：
   - 用戶名：`testuser`
   - 密碼：`testpass123`

## 功能測試

### 1. 查看儀表板

登入後會看到：
- 總持倉價值
- 跟單比例和設定
- Master 倉位
- 我的倉位
- 最近交易記錄

### 2. 測試 Master 下單

1. 點擊右下角的閃電圖標（⚡）
2. 在測試控制台中設定參數：
   - 交易對：BTC/USDT
   - 倉位大小：1.0
   - 開倉價格：50000
3. 點擊「模擬 Master 下單」
4. 觀察儀表板數據變化：
   - Master 倉位立即更新
   - 3 秒後跟隨者倉位更新

### 3. 查看實時更新

儀表板每 5 秒自動刷新，你會看到：
- 引擎狀態（Running/Stopped）
- 最新的交易記錄
- 倉位變動

## 常見問題

### Q1: npm install 失敗

**A**: 確認 Node.js 版本 >= 16：
```powershell
node --version
```

如果版本過舊，請從 https://nodejs.org/ 下載最新版本。

### Q2: 無法連接到後端

**A**: 檢查：
1. 後端是否運行：`docker ps`
2. 後端健康檢查：訪問 http://localhost:8000/health
3. CORS 設定是否正確

### Q3: 登入失敗

**A**: 確認：
1. 後端資料庫中是否有測試用戶
2. 用戶名和密碼是否正確
3. 查看瀏覽器 Console 中的錯誤訊息

### Q4: 看不到數據

**A**: 
1. 確認已登入（檢查 localStorage 中的 token）
2. 打開瀏覽器開發者工具查看 Network 標籤
3. 檢查 API 請求是否成功
