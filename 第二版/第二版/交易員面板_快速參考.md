## 📊 功能速查

### 統計卡片（頂部）
| 卡片 | 說明 | 顏色 |
|------|------|------|
| 總客戶數 | 所有客戶總數 | 灰色 |
| 啟用中 | 正在跟單的客戶 | 綠色 |

| 總資產 | 所有客戶資產總和 | 藍色 |

```
搜尋框: 輸入客戶名稱或郵箱
```

### 客戶表格列
| 列名 | 說明 |
|------|------|
| 客戶資訊 | 名稱 + 郵箱 |

| 最後更新 | 時間戳 |
| 操作 | 管理按鈕 |

## 🎨 狀態標籤

### 顏色系統
```
🟢 啟用中 (active)
   - 背景: 淺綠色
   - 文字: 深綠色
   - 圓點: 綠色脈動

🟡 待審核 (pending)
   - 背景: 淺黃色
   - 文字: 深黃色
   - 圓點: 黃色脈動

🔴 已封鎖 (blocked)
   - 背景: 淺紅色
   - 文字: 深紅色
   - 圓點: 紅色脈動
```

## 🛠️ 操作按鈕

### 可用操作
| 當前狀態 | 可用操作 |
|----------|----------|
| 待審核 | 啟用、封鎖 |
| 啟用中 | 封鎖 |
| 已封鎖 | 解除封鎖 |

### 操作效果
```
啟用 → 狀態變為 "啟用中" (綠色)
封鎖 → 狀態變為 "已封鎖" (紅色)
解除封鎖 → 狀態變為 "待審核" (黃色)
```

## 📊 統計計算

### 自動計算
```javascript
總客戶數 = clients.length
啟用中 = clients.filter(c => c.status === 'active').length
待審核 = clients.filter(c => c.status === 'pending').length
已封鎖 = clients.filter(c => c.status === 'blocked').length
總資產 = clients.reduce((sum, c) => sum + c.totalValue, 0)
```

## 🐛 常見問題

### Q: 頁面白屏？
```
A: 檢查容器是否啟動
   docker ps | grep frontend
   docker logs ea_trading_frontend
```

### Q: 數據不顯示？
```
A: 當前使用 Mock 數據
   刷新頁面會重置
```

```
A: 檢查搜尋詞拼寫
```
